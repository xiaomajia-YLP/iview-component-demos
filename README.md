# vue

## 项目启动

```
# Project setup
npm install

# Compiles and hot-reloads for development
npm run serve

# Compiles and minifies for production
npm run build

npm run test
# Run your tests

# Lints and fixes files
npm run lint

# 进入图形化管理界面
vue ui
```
